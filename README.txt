= Automatic installation (easiest way) =

To do an automatic install of WG Responsive Slider, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "WG Responsive Slider" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".

= Manual installation =

**Uploading in WordPress Dashboard**

1. Download `wg-responsive-slider.zip`
2. Navigate to the 'Add New' in the plugin dashboard
3. Navigate to the 'Upload' area
4. Select `wg-responsive-slider.zip` from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard


**Using FTP**

1. Download `wg-responsive-slider.zip`
2. Extract the `wg-responsive-slider` directory to your computer
3. Upload the `wg-responsive-slider` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

The WordPress codex contains [instructions on how to install a WordPress plugin](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
